<h1>SIGAA - Ferramentas de Utilidade</h1>
<p>Este add-on foi criado para reunir diversas funcionalidades úteis para melhorar a experiência do usuário nos sites do sistema SIGAA.</p>
<p>Atualmente, essas funcionalidades incluem um sinalizador de conflitos de horário para o período de matrícula e rematrícula, e um calculador de total de horas de estágio
 para quando o estudante for realizar o pré-cadastro de estágio. </p>
 <h2> Como utilizar?</h2>
 <p>Primeiramente é preciso baixar o add-on, que foi criado para firefox desktop. Se estiver disponível, estará na minha página do Mozilla:
 <a href="https://addons.mozilla.org/pt-BR/firefox/user/19723052/">https://addons.mozilla.org/pt-BR/firefox/user/19723052/</a></p>
 <p>Alternativamente, o usuário pode instalar o add-on como temporário, indo em about:debugging, meu firefox, carregar add-on temporário, e selecionando o manifest.json dentro da pasta.</p>
